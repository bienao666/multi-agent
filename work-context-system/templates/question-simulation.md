# Question Simulation Template

这个模板用于让 AI 模拟不同读者看到材料后可能提出的问题。

## 输入

```text
请读取：
- .work-context/people/<人物画像>.md
- .work-context/projects/<项目>.md
- .work-context/events/<相关事件>.md
- 当前草稿或报告

请基于读者画像模拟对方可能追问的问题。
```

## 输出格式

```md
# 追问模拟：<材料主题>

## 读者画像摘要

- 角色：
- 关注点：
- 表达偏好：
- 忌讳点：

## 可能追问

| 序号 | 可能问题 | 为什么会问 | 建议回答 | 是否补充到正文 |
| --- | --- | --- | --- | --- |
| 1 |  |  |  | 是 / 否 |
| 2 |  |  |  | 是 / 否 |
| 3 |  |  |  | 是 / 否 |

## 正文补强建议

- 建议 1
- 建议 2
- 建议 3

## 风险提醒

- 哪些问题现在还回答不了。
- 哪些数据需要补充。
- 哪些说法需要更谨慎。
```

## 使用提示词

```text
请基于指定人物画像和当前草稿，模拟对方最可能追问的 8 个问题。
要求：
1. 每个问题说明对方为什么会问。
2. 给出建议回答。
3. 判断是否应该提前补充到正文。
4. 标出目前回答不充分、需要补数据的问题。
```
